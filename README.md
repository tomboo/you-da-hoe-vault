# you-da-hoe Vault

Obsidian vault for the **you-da-hoe** novel project.

## Structure

- **Library/scenes/** - Read-only archive from depot (566 unique scenes)
- **Manuscript/scenes/** - Active work, editing and new scenes
- **Research/** - World-building, character notes, reference material
- **Planning/** - Outlines, templates, session notes

## About

This vault contains content sourced from the [firebase-depot](https://github.com/tomboo/firebase-depot) archive system. The Library folder provides reference to all previously written scenes, while the Manuscript folder is where active writing and editing occurs.

**Status:** Prototype phase - testing vault workflow

## Git Workflow

This vault uses automatic commits to preserve work:
- Changes are version controlled
- Git history tracks manuscript evolution
- GitHub provides off-machine backup
